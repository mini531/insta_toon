구성: 슬라이드 10장, 각 장 안에 2~3개 장면을 세로로 적층 / 플랫 저채도 화풍 / 텍스트는 그림과 분리해 합성

제작 가이드

  • 장 수: 총 10장(슬라이드). 한 장 안에 여러 컷을 위아래로 짜임새 있게 배치.
  • 화풍: 플랫 2D 카툰, 저채도 파스텔, 가는 갈색 라인, 음영 최소. 캐릭터 시트 고정. 반실사/수채 금지.
  • 두 노인 구분: 시아버지 = 브라운 카라 스웨터 / 친정아빠 = 회갈색 카라셔츠 위 스웨터.
  • 생성 제약: 1:1(1024x1024)만 가능 -> 생성 후 좌우를 잘라 4:5로 센터 크롭. 인물과 핵심 요소는 중앙 4:5 안전영역에, 좌우 여백은 잘려도 되게. 장면 적층(상/하)은 좌우 크롭에 안전.
  • 텍스트(분리 합성): 그림에는 글자 없이 여백만. 나레이션 = 상/하단 가로 띠, 대사 = 화자색 말풍선. 빈 말풍선이 아니라 합성용 빈 공간을 둠.
  • 워터마크: '@toon_observer_' 를 이미지마다 임의 위치 2곳에 옅고 흐리게.
