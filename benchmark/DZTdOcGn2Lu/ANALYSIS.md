## 1. 벤치마크 핵심 분석

### 1-1. 그림체
- 극단적으로 단순한 플랫 카툰. 캐릭터가 토끼/곰 같은 동물형 단순체 (눈=점, 코·입=짧은 선).
- 외곽선: 가늘고 부드러운 갈색 톤. 검정 강선·음영 거의 없음.
- 채색: 저채도 파스텔 플랫 채색, 그림자 최소.
- 배경: 대부분 종이질감 단색 또는 아주 단순한 소품 1~2개. 디테일을 의도적으로 비움.

### 1-2. 컷 구성 - "한 컷 = 2~3 장면" (사용자가 지적한 핵심)
- panel_03 ~ panel_10 대부분이 **한 장(슬라이드) 안에 위/아래로 2개의 미니 장면**을 쌓은 구조.
  - 예) panel_03: (상단) 공원에서 회사동료와 마주침 + 인사 말풍선 → (하단) 대화가 이어지며 "임신했어?" 클리프행어.
  - 예) panel_04: (상단) 당황한 반응 → (하단) 동료의 변명/오지랖.
  - 예) panel_05: (상단) 남편과 대화 → (하단) "40대잖아" 핀잔.
- 장면 사이는 **배경 색 띠를 바꿔(베이지↔연보라↔연녹)** 자연스럽게 분리. 명시적 칸 테두리는 거의 안 씀.
- 즉 1슬라이드가 사실상 2~3컷짜리 웹툰 스트립. 그래서 4:5보다 세로로 길게 읽히고 정보 밀도가 높음.

### 1-3. 텍스트 처리
- **나레이션**: 장면 위/아래에 가로 텍스트로 배치(말풍선 아님). 흰 배경 띠 위 검정 글씨.
- **대사**: 둥근 말풍선. 화자 색상으로 구분(주인공=하늘/민트, 상대=베이지/분홍).
- 표지(panel_01)는 큰 손글씨 제목 + 짧은 한 줄 후킹. 제품/브랜드 로고는 상단에 작게.
- 글자는 항상 **이미지에 직접 렌더링**되어 있고 빈 말풍선이 없음.

### 1-4. 비율/포맷
- 슬라이드 자체는 세로(약 4:5~3:4). 내부 2장면 적층으로 체감 세로비 더 김.
- 10컷이 하나의 흐름(도입-갈등-반전-광고전환-마무리)으로 연결.

## 2. ep01(기제작 10장) 문제점 분석

> ep01/01~10.png, 전부 **1024x1024 정사각형(1:1)**, AI 일러스트체.

### 문제 A. 비율이 규격 위반 (1:1)
- AGENTS.md 목표는 **4:5 세로(인스타 최적화)** 인데 전부 1:1 정사각형.
- 벤치마크처럼 세로로 길지 않아, "한 컷 2장면" 구성을 담을 세로 공간 자체가 없음.

### 문제 B. 그림체가 벤치마크와 완전히 다름 + 컷마다 화풍이 흔들림
- 벤치마크: 초단순 플랫 동물체 / ep01: 디테일한 반(半)실사 인물체 (음영·머리결·주름 묘사).
- 더 큰 문제는 **패널 간 화풍 불일치**:
  - 01·02·06·07: 또렷한 갈색 라인 + 플랫에 가까운 카툰.
  - 05: 수채 느낌의 부드러운 일러스트(라인 약함).
  - 10: 거의 애니/반실사 톤(음영·하이라이트 강함, 볼터치).
  - 08·09: 라인 두께·질감이 또 다름.
- 즉 10장이 "같은 작품의 같은 화가" 로 안 보임.

### 문제 C. 캐릭터 일관성 붕괴 (가장 치명적)
- **아내**: 05·10에선 갈색 긴 웨이브 + 또렷한 이목구비, 06·08에선 라인/얼굴형이 다르고, 07에선 또 다른 얼굴. 같은 인물로 식별 불가.
- **친정엄마**: 02에선 검정 단발인데 09에선 얼굴 인상이 달라짐.
- **시아버지/친정아빠**: 둘 다 대머리+옆머리 노인이라 서로 구분이 안 됨(02의 노인=친정아빠인데 시아버지와 혼동).
- **레퍼런스 시트는 이미 존재함**(`.agents/character_sheet.png`, 6인 전신). 그런데 ep01 생성이 이 시트를 따르지 않음(특히 10컷 아내=반실사로 시트와 완전 다름). → 시트를 실제 참조로 넣지 않고 매 컷 독립 생성한 것이 원인.
- 두 노인 구분 기준(시트 기준): **시아버지=브라운 카라 스웨터 / 친정아빠=회갈색 카라셔츠 위 스웨터**. ep01은 이를 안 지켜 혼동됨.

### 문제 D. "한 컷 1장면" 만 만듦 (벤치마크 핵심을 미반영)
- ep01은 10컷 모두 단일 장면 1개. 벤치마크의 "한 슬라이드 안 2~3장면 적층"이 전혀 없음.
- 사용자 지적 그대로: 벤치마크는 한 컷에 두세 장면이 같이 들어가 정보·리듬이 살아있는데 ep01은 밋밋함.

### 문제 E. 말풍선/텍스트 미처리
- 01·07·08: **빈 말풍선**만 그려지고 대사가 안 들어감(텍스트 미렌더). 벤치마크는 항상 글자가 박혀 있음.
- 나레이션 띠도 없음. 스토리에 있는 대사/나레이션이 이미지에 반영되지 않아 단독으로 의미 전달이 안 됨.

### 문제 F. 장면-스토리 불일치 (오생성)
- **08컷**: 지문은 "시아버지는 TV만, 남편은 휴대폰만 - 둘 다 무관심/침묵". 그런데 생성물은 시아버지가 아내에게 다가가 **말풍선까지 띄우며 대치**. TV 없음. 나레이션(침묵)과 정반대.
- 06·09 등은 의도와 비슷하나, 배경 소품(액자·화분·전화기·러그)이 과하게 사실적으로 들어가 벤치마크의 미니멀 톤과 어긋남.

### 문제 G. 배경 과밀
- 벤치마크는 배경을 단색 띠로 비우는데, ep01은 거실 풍경을 꽉 채워(가구·액자·식물·바닥결) 시선이 분산되고 인물·대사 가독성이 떨어짐.

## 3. 개선 방향 (다음 작업 제안)
1. **캔버스 4:5 세로**로 변경(예: 1080x1350). 또는 벤치마크식 더 긴 세로.
2. **한 컷에 2~3 미니 장면 적층** 구조 채택. 장면 분리는 배경 색 띠로. (스토리 10컷을 2장면씩 묶으면 슬라이드 수도 자연스럽게 정리됨)
3. **그림체 1종 고정**: 플랫·저채도·가는 갈색 라인·음영 없음. 반실사/수채 금지.
4. **캐릭터 레퍼런스 시트 1장 먼저 제작** → 모든 컷에서 동일 시트를 참조(동일 헤어·의상·얼굴). 노인 2명은 의상색(시아버지=브라운, 친정아빠=회갈색)·헤어로 확실히 차별화.
5. **대사/나레이션을 이미지에 직접 렌더링**. 빈 말풍선 금지, 나레이션은 상/하단 흰 띠 텍스트.
6. **배경 미니멀화**: 단색/소품 1개. 가독성 우선.
7. 08컷처럼 스토리와 어긋난 장면은 지문 충실히 재생성(침묵=말풍선 없음, TV·휴대폰 명시).
